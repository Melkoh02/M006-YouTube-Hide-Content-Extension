# YT Cleaner

A lightweight browser extension that hides **Members-only** videos and **Shorts** from YouTube channel pages.  
Designed to declutter the Videos tab without modifying your YouTube account or data.

### Features
- Toggle hiding *Members-only* videos
- Toggle hiding *Shorts*
- Works automatically on channel `/videos` pages
- No data collection, no external requests, zero tracking
